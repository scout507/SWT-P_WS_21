◆ Update()

void FenceInteraction.Update ( )
private

Executed by client and server. When a player enters the action radius, he receives an output. When he presses [E], he activates a command to activate the repair. The server repairs the fence only when a player starts this. The action is cancelled when the player performing the action moves away. In addition, it is always checked whether a new board is broken, otherwise it is added to the "brokenPlanks" list.
